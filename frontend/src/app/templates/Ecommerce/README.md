# ProductListing Component

A customizable product grid component for the design editor.

## Style

### Default Design
- Clean, modern product cards with hover effects
- Product image with zoom animation on hover
- Card-based layout with customizable spacing
- Professional typography hierarchy
- Responsive grid (1-4 columns)

### Visual Elements
- Product image (with object-cover)
- Product name (bold, 20px)
- Product description (optional, 14px gray text)
- Price display (large, custom color)
- "Add to Cart" button (full-width)

### Color Scheme
- White card backgrounds
- Gray text for descriptions
- Customizable accent colors
- Button variants: primary (blue), secondary (green), outline

## Customizable Style Options

### Layout
- Grid columns: 1, 2, 3, or 4
- Column gap spacing (px)
- Card height (auto or fixed px)
- Image height (100-600px)

### Colors
- Section background
- Title text color
- Price accent color
- Button style

### Card Styling
- Border radius (0-50px)
- Shadow effects (on/off)
- Show/hide descriptions

## Files

- **ProductListing.tsx** - Component
- **ProductListingSettings.tsx** - Settings panel
- **index.ts** - Exports
